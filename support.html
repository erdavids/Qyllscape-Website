Support

While Qyllscape is designed to be simple, these manuals attempt to help answer any questions that might come up as well as highlight features that users may overlook. Furthermore, there is information about the file system to help users understand how their writing will be organized and saved.

If you encounter questions related to Qyllscape that are not addressed by the manual, or find something that would be considered an issue, please reach out to the following email:

thebuffed [at] gmail.com

Please keep in mind that Qyllscape development will continue after release, and features can be added, changed, or improved. Please check out the Updates Page for more information about ongoing development.
