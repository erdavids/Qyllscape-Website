This page will be attempt to keep users up to date on any updates related to fixing issues or adding/improving features. Manuals for each version can be found by visiting the Support Page.

Planned update - Qyllscape 1.1.0

The initial release of Qyllscape was intended to offer the core features necessary for writing. Now the fun part will begin as I start to work on improving existing features, and adding things that I wasn't able to for the first version. Here is a quick look at my priorities for the first major update:

  • Introduce placeable stickers, allowing users to 'decorate' their writing environment with aesthetic graphics.
  • Allow users to upload their own graphical content (Textures, Book Designs, Stickers, etc.) and share with others through Steam Workshop.
  • Improve the exporting capabilities. Right now only a basic export is enabled, but in the future, users will be able to customize their exports to make their manuscript as ready as possible to share.
Qyllscape 1.0.6 - August 29, 2024
  • Added ability for user to upload background images.
  • Added ability for user to upload fonts.
  • Added workaround for rich-text, shortcuts for adding italics, bold, and underline html tags.
  • Added a quick search tool that will open settings matching certain keywords.
  • Split Font and Page Details into own settings page.
Qyllscape 1.0.5 - April 27, 2024
  • Added option under 'General Settings' to toggle fullscreen
  • Fixed various alignment issues.
  • Added tool for automatically importing previously created backup projects
  • Added a simple project details page to show total word count (More analytics are being developed)
Qyllscape 1.0.1 - Feb. 28, 2024
  • Added option under 'General Settings' to persist more theme data (candlelight, single page, minimalist mode)
  • Fixed issue with showing placeholder text for right page when in single page mode
  • Fixed issue with candlelight automatically turning off lights 2-4 when opening tool
  • Choice for 'Show Details' (titles/page numbers) now persists
  • Slowed down single page camera animation to be less sudden
  • Added clarity to candlelight tool with default positions, better instructional text
Qyllscape 1.0.0 - Feb. 26, 2024
  • Initial release of Qyllscape containing first iteration of all base features.
